AwesomeWM Config

A custom AwesomeWM configuration (rc.lua) that integrates all LinuxUtilities tools into a cohesive desktop environment — unified hotkeys, a system-tray widget, and Antsand-branded theming.

Installation

# Back up existing config
cp ~/.config/awesome/rc.lua ~/.config/awesome/rc.lua.bak

# Install from LinuxUtilities
cd LinuxUtilities
cp rc.lua ~/.config/awesome/rc.lua

# Reload AwesomeWM (no restart needed)
# Press Mod4+Ctrl+r  or:
awesome-client 'awesome.restart()'
Note: The config expects all LinuxUtilities binaries to be in $PATH or in ~/Workspace/LinuxUtilities/. Adjust paths in the -- tool paths section near the top of rc.lua.

Crash Recovery

If AwesomeWM crashes or fails to load the config:

  1. Press Ctrl+Alt+F2 to switch to a TTY
  2. Log in and restore the backup: cp ~/.config/awesome/rc.lua.bak ~/.config/awesome/rc.lua
  3. Press Ctrl+Alt+F7 to return to the graphical session
  4. Reload: awesome-client 'awesome.restart()'

Alternatively, the rescue-windows.sh script can recover floating windows if layouts break.
