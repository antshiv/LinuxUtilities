One-Line Install

sudo apt install build-essential pkg-config \
    libx11-dev libxext-dev libxfixes-dev libxrender-dev libcairo2-dev \
    libgtk-3-dev gromit-mpx xdotool
./install_gromit_profile.sh

Building Tools

Cursor Spotlight

./build_cursor_spotlight.sh
./cursor_spotlight --radius 180 --dim 0.68 --fps 50

Press Esc to exit. See Spotlight docs for all flags.

Linux Control Center

./build_linux_control_center.sh
./linux_control_center

GTK3 app with Night Light, Shortcuts, and Screenshots tabs.

GTK DSL Demo

./build_gtk_dsl_demo.sh
./gtk_dsl_demo dsl/workbench.gdsl

Edit dsl/workbench.gdsl to customise the launcher.

Presenter Dash

# No build step — it's a shell script
./install_gromit_profile.sh   # install shape/modifier map once
./presenter_dash.sh anchor   # set anchor at cursor
./presenter_dash.sh dash     # dashed segment
./presenter_dash.sh arrow    # arrow segment

AwesomeWM Configuration

ℹ
Always back up your existing ~/.config/awesome/rc.lua before replacing it.
cp rc.lua ~/.config/awesome/rc.lua

# Reload AwesomeWM
# Press Mod4 + Ctrl + R   (or restart from the menu)

The config adds a Utilities widget to the wibar with buttons for Spotlight, Gromit, Control Center, and shortcuts. All presenter-related hotkeys are on the F6–F11 row.

Building the Docs

cd docs
bash build.sh

# Preview locally
cd build
python3 -m http.server 8080
# Open http://localhost:8080

Generated pages go into docs/build/ which is gitignored. Only _pages/, _partials/, css/, js/, and assets/ are tracked in version control.
