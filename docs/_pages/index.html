Quick Start

1 — Clone & Build

git clone https://github.com/antshiv/LinuxUtilities.git
cd LinuxUtilities

# Cursor spotlight
sudo apt install pkg-config libx11-dev libxext-dev libxfixes-dev libxrender-dev libcairo2-dev
./build_cursor_spotlight.sh
./build_cursor_spotlight.sh --install   # optional: ~/Programs/bin/cursor_spotlight

# Control center
sudo apt install libgtk-3-dev pkg-config
./build_linux_control_center.sh
./build_linux_control_center.sh --install   # optional: ~/Programs/bin/linux_control_center

# Presenter drawing
sudo apt install gromit-mpx xdotool
./install_gromit_profile.sh
./launch_teleprompter.sh

2 — Copy AwesomeWM Config

# Back up your existing config first
cp ~/.config/awesome/rc.lua ~/.config/awesome/rc.lua.backup

# Copy the LinuxUtilities rc.lua
cp rc.lua ~/.config/awesome/rc.lua

# Restart AwesomeWM: Mod4+Ctrl+R

3 — Try It

# Toggle spotlight
./build/bin/cursor_spotlight --radius 180 --dim 0.68 --fps 50

# Launch control center
./build/bin/linux_control_center

# Start presenter drawing (F6 in AwesomeWM)

Philosophy

⚡
Zero bloat. No Electron, no npm, no Python GUI frameworks. Everything is C + GTK3, Bash, or Lua — tools that are already on your system or one apt install away.
