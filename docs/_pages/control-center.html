Control Center

A native GTK3 desktop utility that consolidates frequently used desktop controls into a single tabbed window. Built in C with no runtime dependencies beyond GTK3.

Build & Run

cd LinuxUtilities
./build_linux_control_center.sh   # compiles linux_control_center.c
./linux_control_center              # launch the app

Or compile manually:

gcc -o linux_control_center linux_control_center.c \
    $(pkg-config --cflags --libs gtk+-3.0) -lm

Features

🌙 Night Light

Adjusts screen color temperature using redshift. Toggle between warm (3500K) for evening and neutral (6500K) for daytime — easy on the eyes during long sessions.

Warmredshift -O 3500
Neutralredshift -x

📸 Screenshots

Quick access to common screenshot workflows — full screen, active window, or region selection. Saves to ~/Screenshots/ with timestamps.

⌨️ Shortcuts Viewer

In-app reference for all AwesomeWM keybindings. Displays the same shortcut tables from the Shortcuts docs page in native GTK widgets.

🔊 Audio Controls

PulseAudio / PipeWire volume sliders with per-application mixing. Switch outputs (speakers ↔ headphones) without opening pavucontrol.

📝 Teleprompter Launch

Utilities tab includes a one-click teleprompter launcher for ATEM workflows where laptop panel shows script and HDMI feed stays clean.

🧭 Presenter Canvas

Open a dedicated browser whiteboard for live drawing with pen/shape/icon tools, animated flow lines, and JSON save/load for reusable episode scenes.

🎬 Storyboard DSL

Launch a timeline parser/player to script bytebytego-style step-by-step animation from JSON DSL. Includes C-Kernel-Engine train/infer presets and canvas-import generation.

Configuration

The control center reads its configuration from the C source at compile time. To customize:

SettingLocation in SourceDefault
Night Light warm tempNIGHT_TEMP_WARM3500K
Screenshot save dirSCREENSHOT_DIR~/Screenshots
Window sizegtk_window_set_default_size600×400

AwesomeWM Integration

Launch the Control Center from AwesomeWM with a hotkey:

-- rc.lua
awful.key({ modkey }, "c", function()
    awful.spawn("linux_control_center")
end, {description = "open control center", group = "utilities"})
