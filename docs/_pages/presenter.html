Presenter Flow

Real-time drawing overlays and animated path segments for live presentations. Combines Gromit-MPX for freehand drawing with a custom presenter_dash.sh for structured flow diagrams.

Setup

sudo apt install gromit-mpx xdotool
./install_gromit_profile.sh   # installs compat profile by default

Both tools are controlled via AwesomeWM hotkeys defined in rc.lua. The installer writes ~/.config/gromit-mpx.cfg and auto-selects a parser-safe profile for distro Gromit builds (for example Ubuntu's 1.5.x package).

Presenter Dash (Animated Segments)

The presenter_dash.sh script draws animated line segments between an anchor point and the current cursor position using xdotool for coordinates.

Workflow

1. Position cursor where you want the start point 2. Alt+F11 → sets the anchor 3. Move cursor to the end point 4. F11 → draws a dashed segment from anchor → cursor (anchor advances to new cursor position automatically) 5. Repeat step 3-4 to chain segments together

Segment Styles

KeyStyleUse Case
F11── ── ── (dashed)Data flow, process steps
Shift+F11· · · · · (dotted)Optional paths, dependencies
Ctrl+F11────── (solid)Strong connections
Mod4+F11───▶ (arrow)Direction, causality

CLI Usage

./presenter_dash.sh anchor   # Set start point
./presenter_dash.sh dash     # Draw dashed segment
./presenter_dash.sh dot      # Draw dotted segment
./presenter_dash.sh solid    # Draw solid segment
./presenter_dash.sh arrow    # Draw arrow segment
./presenter_dash.sh clear    # Clear all drawings
./presenter_dash.sh undo     # Undo last stroke
./presenter_dash.sh redo     # Redo stroke
./presenter_dash.sh reset    # Remove anchor point

Presenter Canvas + Storyboard DSL

For slide-like scene animation (instead of drawing over existing apps), use the browser canvas + timeline pair:

./launch_presenter_canvas.sh
./launch_presenter_storyboard.sh

The Storyboard player includes ready-made CKE Train Preset and CKE Infer Preset templates to accelerate LLM training/inference video production.

Timeline + Transcript Workflow

The canvas now includes a Timeline / Transcript panel so you can play timed captions during narration.

  1. Open ./launch_presenter_canvas.sh.
  2. In Timeline / Transcript, click Load Transcript JSON.
  3. Optionally click Load Audio to sync timeline to voice track.
  4. Press K (or Play) to run timeline and show live caption overlay.
  5. Use Add Caption At Time to insert the current segment as editable canvas text.

Transcript JSON can be { "segments": [...] } or a top-level array where each segment has start, end, and text.

The bottom timeline editor includes caption clip drag/trim/split, multi-select (Shift+click), snap presets, and zoom for precise narration timing.

Recent UI updates: collapsible sidebar sections keep controls compact, tool/action buttons are icon-labeled, and caption presets render as quick style cards with short descriptors.

Pipeline

1. Draw your scene in Presenter Canvas 2. Export starter DSL with Ctrl+Shift+S 3. Open Storyboard DSL Player, load/edit timeline actions 4. Press Space to play and record narration in one take

Supported DSL actions: show, hide, draw, move, pulse, zoom, pan. Targets can be exact ids, *, or selectors such as type:edge.
